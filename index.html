Como usar
1
Clique em Registrar toda semana e avalie suas 8 categorias com estrelas de 1 a 5 (clique na metade esquerda da estrela para meia nota).
2
Adicione metas específicas para a semana e notas de texto livre. Você marca as metas como concluídas no Dashboard.
3
O sistema converte sua média em pontos de 0 a 50. A meta é 40 pontos (= média 4.0★). Uma mensagem de feedback é gerada automaticamente.
4
O Dashboard mostra gráfico radar, barras por categoria, tendência em relação à semana passada e o histórico de evolução.
5
O Calendário marca as semanas registradas. Clique em qualquer dia com ponto azul para ver o resumo daquela semana.
6
Use Notas para anotações livres que não são necessariamente de uma semana específica — ideias, reflexões, listas.

Onde ficam os dados?

No Supabase (nuvem). Acessível de qualquer dispositivo com sua conta. Se offline, salva localmente como backup.

Como funciona a meia estrela?

Clique na metade esquerda de uma estrela para a nota X.5. Metade direita para nota X cheia. O label ao lado mostra o valor.

Sistema de pontos

Pontos = (média ÷ 5) × 50. Exemplos: 5★ = 50pts, 4★ = 40pts (meta), 3★ = 30pts, 2★ = 20pts.

Registrar de novo?

Você pode editar a semana atual abrindo o Questionário novamente. A semana é identificada pelo número ISO — sem duplicatas.
